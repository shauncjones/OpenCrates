## Issue reporting
Please include the following:

* Minecraft version
* Simple Storage version
* Forge version/build
* List of other mods & versions
* Screenshots are appreciated 
* For crashes:
	* Steps to reproduce
	* ForgeModLoader-client-0.log (the FML log) from the root folder of the client
